ロールオーバープラグイン

画像にhoverした時に、imgの画像を差し替えるのがロールオーバープラグインです

参照：input要素にも使えるjQueryのロールオーバープラグイン - Rewish

Script

jQuery(function($){
	$('.rollover').rollover();
});

CSS


.rollover-demo {
	background-color: #ccc;
}

HTML

<div id="demo" class="mb100">
	<p class="rollover-demo"><a href="#"><img src="../img/ham.png" class="rollover"></a></p>
</div>
