アコーディオン機能 ステップ５（実演）

仕様を変える：マルチオープンをシングルオープンに

  • 初期状態：CSSでコンテンツは非表示
  • 見出しをクリック
  • 見出しに開いていることがわかるようにアクティブスタイル追加
  • 開いているところを閉じる
  • 見出しの次の要素が縦エフェクトで開く（何度も開かないようにする）

Script

jQuery(function($){
	$('#accordion dt').on('click', function () {
		if($(this).hasClass('is-active')) {
			$(this).removeClass('is-active').next().stop().slideUp();
		} else {
			$('.is-active').removeClass('is-active').next().stop().slideUp();
			$(this).toggleClass('is-active').next().stop().slideToggle();
		}
	});
});
