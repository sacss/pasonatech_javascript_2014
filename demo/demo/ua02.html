スマートフォンかどうか

ユーザーエージェントの分岐を使って、スマートフォンかどうかの分岐をします

Script

jQuery(function($){
	var ua = navigator.userAgent;
	$('button').on('click', function () {
		if(ua.indexOf('iPhone') > 0 || ua.indexOf('iPod') > 0 || ua.indexOf('Android') > 0) {
			alert('スマートフォンです');
		} else {
			alert('いいえ、PCです');
		}
	});
});

HTML

<button>ユーザーエージェント</button>
