タブ機能 ステップ１

どのような動きをするのか仕様を考える

まずはシンプルなところから

  • 初期状態：CSSでコンテンツは非表示
  • コンテンツの最初を開く
  • タブをクリックする
  • コンテンツを一度全部非表示
  • aタグに仕込んでいたハッシュを持つIDの（連動する）コンテンツが開く

Script

jQuery(function($){
	$('.tab__content:first').show();
	$('.tab__button a').on('click', function () {
		$('.tab__content').hide();
		$($(this).attr('href')).show();
	});
});

HTML

<div id="tab">
  <ul class="tab__list">
    <li class="tab__button"><a href="#tab1">タブ１</a></li>
    <li class="tab__button"><a href="#tab2">タブ２</a></li>
    <li class="tab__button"><a href="#tab3">タブ２</a></li>
  </ul>
  <div id="tab1" class="tab__content">タブ１</div>
  <div id="tab2" class="tab__content">タブ２</div>
  <div id="tab3" class="tab__content">タブ３</div>
</div>

CSS

#tab {
	font-size: 24px;
	margin: 5px 10px;
}
.tab__list {
	padding: 0;
	overflow: hidden;
}
.tab__button {
	float: left;
}
.tab__button a {
	display: block;
	padding: 10px 20px;
	border: solid #34495e;
	border-size: 5px 5px 0 5px;
}
.tab__button a:hover {
	color: #fff;
	background-color: #34495e;
}
.tab__content {
	display: none;
	width: 100%;
	height: 300px;
	color: #222222;
	border: 10px solid #34495e;
}
