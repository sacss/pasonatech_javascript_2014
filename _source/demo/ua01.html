ブラウザ分岐 ユーザーエージェント

ブラウザで処理

ブラウザが持っている、そのブラウザは、何？という情報をJavaScriptでとれる。

Script

jQuery(function($){
	var ua = navigator.userAgent;
	$('button').on('click', function () {
		alert(ua);
	});
});

HTML

<button>ユーザーエージェント</button>
