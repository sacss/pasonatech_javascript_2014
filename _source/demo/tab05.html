タブ機能 ステップ５（実演）

コードの整理

  • 何度も出てくるセレクタは変数に置き換える
  • 初期状態：CSSでコンテンツは最初を除いて非表示（classで制御）
  • コンテンツの最初を開く
  • タブをクリックする
  • 既にアクティブなら処理をしないで終了
  • タブのアクティブを一度除去
  • クリックしたタブにclassをつけてアクティブに、aタグが動かないようにする
  • コンテンツを一度全部非表示（classで制御）
  • aタグに仕込んでいたハッシュを持つIDの（連動する）コンテンツが開く（classで制御）

Script

jQuery(function($){
	var $tab__button = $('.tab__button');
	$tab__button.find('a').on('click', function (e) {
		e.preventDefault();
		var $this = $(this);
		var $thisParent = $this.parent();
		if (!$thisParent.hasClass('is-active')) {
			$tab__button.removeClass('is-active');
			$thisParent.addClass('is-active');
			$('.tab__content').removeClass('is-active');
			$($this.attr('href')).addClass('is-active');
		}
	});
});
