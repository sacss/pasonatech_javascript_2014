プラグイン関連 1 プラグインの使い方

プラグインとは

jQuery の本体は、JavaScript を簡単に扱うためのメソッドが詰まったライブラリですが、jQuery 単体ではその機能を準備するだけ、動かしたりなどはしてくれません。

それをさらに機能くわえたり、特定の動きの指示を予めいれて要素だけを指定するだけにしたようなものを「プラグイン」といいます。

ここまで解説してきたタブやアコーディオン機能なども、HTMLとclassと、僅かなスクリプトを書くだけで使えるようにしてくれたりするものもあります。

ここではそんなプラグインの使い方を簡単に解説します。

プラグインの基本的な使い方

読み込み場所

jQueryの本体を読み込んだあとにプラグインを指定して読み込み、その後にプラグインを使うためのコードをかきます。

<script src="js/jquery-2.0.3.min.js"></script>
<script src="js/plugin.js"></script>

利用例：jQuery Color

参照：jquery/jquery-color

読み込み

<script src="js/jquery-2.0.3.min.js"></script>
<script src="js/jquery.color-2.1.2.min.js"></script>

Script

jQuery(function($){
	$('button.white_effect01').on('click', function () {
		var $demoBlock = $('.demo__block');
		if(!$demoBlock.hasClass('is-white')) {
			$('.demo__block--type1').stop().animate({
				backgroundColor: '#fff'
			}, 1000).addClass('is-white');
		} else {
			$('.demo__block--type1').stop().animate({
				backgroundColor: '#000'
			}, 1000).removeClass('is-white');
		}
	});
});

CSS

.demo__block--type1 {
	width: 100%;
	height: 200px;
	background-color: #000;
}

HTML

<div id="demo" class="mb100">
<button>色変更</button>
<div class="demo__block--type1"></div>
