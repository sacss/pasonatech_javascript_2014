アコーディオン機能 アニメーション演習

仕様を変える：横開閉にする

  • 初期状態：横型スタイルを作る
  • 初期状態：CSSでコンテンツは非表示
  • 見出しをクリック
  • 見出しに開いていることがわかるようにアクティブスタイル追加
  • 開いているところを閉じる
  • 見出しの次の要素が縦エフェクトで開く（何度も開かないようにする）

Script

jQuery(function($){
	$('#accordion dt').on('click', function () {
		$(this).next().animate({
			width: 'toggle'
		}, 'slow');
	});
});
