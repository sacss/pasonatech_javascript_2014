プラグイン関連 3 エフェクトの緩急 1

エフェクトの緩急 1

エフェクトで同じ時間動かすにしても、その緩急のつけかた次第で、全く別の動きに見えるようになります。

まずこのページでは、jQueryのデフォルトで用意されいてる緩急を見てみましょう

linear（リニア）とswing

デフォルトで用意されている緩急（easing）は、linearとslingの２つです。

この２つの違いを見てみましょう

Script

jQuery(function($){
	var i = 0;
	$('button').on('click', function () {
		var mleft = (i % 2)? 0 : '80%';
		$('.default').stop().animate({
			marginLeft: mleft
		}, 2000);
		$('.linear').stop().animate({
			marginLeft: mleft
		}, 2000, 'linear');
		$('.swing').stop().animate({
			marginLeft: mleft
		}, 2000, 'swing');
		i++;
	});
});
