プラグイン関連 7 イメージスライダー（カルーセル）

スライダープラグイン

メインビジュアルなどで複数枚の画像を、左右やフェードなどで切り替えて見せるプラグインです

ユーザビリティー視点でみると１枚目しかほぼ意味無いですが、実装としてよく依頼がきます。

参照：jQuery Content Slider | Responsive jQuery Slider | bxSlider

Script

jQuery(function($){
	$('.bxslider').bxSlider();
});

CSS

.bx-wrapper {
	max-width: 480px !important;
}

HTML

<div id="demo" class="mb100">
	<ul class="bxslider">
		<li><img src="../img/pic1.jpg" /></li>
		<li><img src="../img/pic2.jpg" /></li>
		<li><img src="../img/pic3.jpg" /></li>
		<li><img src="../img/pic4.jpg" /></li>
	</ul>
</div>
