プラグイン関連 4 エフェクトの緩急 2

エフェクトの緩急 2

jQueryのデフォルトで用意されている緩急の種類は少ないのですが、それを補うプラグインが「jQuery Easing Plugin」です

参照：jQuery Easing Plugin

参照：gdsmith/jquery.easing

参照：Easing Function 早見表

デモを通してどのような管球になるか見てみましょう

Script

jQuery(function($){
	var i = 0;
	$('button').on('click', function () {
		var mleft = (i % 2)? 0 : '80%';
		var easingFn = $('#easing').val();
		$('.default').stop().animate({
			marginLeft: mleft
		}, 2000, easingFn);
		i++;
	});
});
