アコーディオン機能 ステップ１

どのような動きをするのか仕様を考える

まずはシンプルなところから

  • 初期状態：CSSでコンテンツは非表示
  • 見出しをクリック
  • 見出しの次の要素が開く

Script

jQuery(function($){
	$('#accordion dd').hide();
	$('#accordion dt').on('click', function () {
		$(this).next().toggle();
	});
});
