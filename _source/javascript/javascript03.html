JavaScript基礎：変数の補足

変数

値を入れておくための入れ物、var を最初につけることで、その後に書かれるキーワードが変数になるという宣言になります。

var a = 'あいうえお';
alert(a); // あいうえお のアラート

文字列、シングルクォーテーションもしくはダブルクォーテーションで囲まれている

変数にしていないものを利用しようとすると、エラーになります。

alert(aaa); // Uncaught ReferenceError: aaa is not defined

varをつけて変数であるという宣言をしておくことで、中身が表示されるようになります。何も代入してなければ 未定義の undefinded になります。

var b;
alert(b); // undefined

functionの中などで宣言された変数はその中でのみ有効になります。

var test;
(function () { // 即時関数
  var test = 'テスト'; // function内のみで有効
  alert(test); // テスト
})();
alert(test); // undefined

注意

var の変数宣言なしで利用すると、グローバル変数になり、上書きされる危険がでてきます。

var test = 'テストです';
alert(test); // テストです
(function () {
  // var なしで宣言で上書きされます
  test = '上書きされます';
  alert(test); // 上書きされます
})();
alert(test); // 上書きされます

必ず変数宣言をして使いましょう。
