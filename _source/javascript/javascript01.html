JavaScript基礎

JavaScript

JavaScript（ジャバスクリプト）は、今回勉強する範囲では主にWebブラウザ上で動作するスクリプト言語。

現在はサーバーサイドでも使われるようになったり（node.js）、用途は様々。

JavaScriptをページで利用するために

JavaScriptを勉強するにあたり、JavaScriptをHTML内で使う方法を先に学んでおきます。

１，HTMLファイルに直接書く

<script>〜</script>

scriptタグ内にJavaScriptのコードを書くことで利用できます。

２，JavaScriptファイルとして読み込む

<script src="js/my_script.js"></script>

基本的にはこちらで利用します。１のHTMLに直接書く方法だと、作業者が分離しにくく、HTMLタグとスクリプトが混在してしまうため、メンテナンス性に欠けてしまうため。

※ しかし、HTMLファイルと別途で読み込みを行うため、読み込み速度面でのみみると１のほうが早いため、ケース・バイ・ケースで使い分けるのが良いです。

早速使ってみる

Google Chrome や Firefox などのブラウザコンソール（開発環境のコンソール）で簡単にコードの実行ができるので試してみましょう

以下のコードを入力してみましょう

alert
alert('アラート');
alert(1 + 1);

実際に入力するところを見ながら覚える基礎

  • プライマリ (プリミティブ) データ型：真偽値（Boolean）、数値（Number）、文字列（String）
  • 複合データ型：オブジェクト（Object）・・・データその物を表す基本的な抽象的な概念 、配列などなど
  • 特別なデータ型：Null、未定義（undefined）
  • 変数
  • 四則計算（+,-,*,/,%・・・数値の足し算、文字列の足し算）
  • 関数（function()など）
  • 繰り返し（forループ）
  • DOM Scripting
