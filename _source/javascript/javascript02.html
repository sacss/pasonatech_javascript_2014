JavaScript基礎：数値と文字列

数値と文字列

数値は半角の数値、計算などで利用可能

1
2
3
10
100
-10

文字列、シングルクォーテーションもしくはダブルクォーテーションで囲まれている

'あ'
'あいうえお'
'abcde'
'12345'

数値の1と文字列の'1'、パッとみて同じにみえても違うもの。

足し算をしてみるとわかる

1 + 1 // 2
'1' + '1' // '11'
1 + '1' // '11'

数値 + 数値 は計算、 文字列 + 文字列 は文字の連結、 数値 + 文字列 は 文字列同士の計算と同じ

補足問題

以下の結果はどうなるか

1 + 1 + '1'
1 + '1' + 1
'1' + 1 + 1
